RU:
Данный код представляет собой реализацию классов для представления футбольных команд, их статистики и отображения этой статистики на экране.

Суть программы:
Дано 14 Греческих команд, каждая из которых сыграла 26 матчей. 
Идёт подсчет статистики каждой из команд отдельно, сколько голов было забито, сколько голов было пропущено, разница в голах, после чего данная статистика выводится в таблицу, выводя отдельно название команды и все ее сыгранные матчи.
В конце программы идет итоговый подсчёт статистики и уже выводятся сами очки, подсчёт которых идет на примере: Победа 3 очка, ничья 1 очко, поражение 0 очков, после идёт вывод таблицы уже каждой команды и количество полученных очков в зависимости от статистики сыгранных матчей и голов.

EN:
This code is an implementation of classes for representing soccer teams, their statistics and displaying these statistics on the screen.

The essence of the program:
Given 14 Greek teams, each of which played 26 matches. 
There is a calculation of statistics for each of the teams separately, how many goals were scored, how many goals were conceded, the difference in goals, then these statistics are displayed in the table, displaying separately the team name and all of its played matches.
At the end of the program there is a final calculation of statistics and the points are already displayed, the calculation of which goes on the example: Win 3 points, draw 1 point, defeat 0 points, then there is a table of each team and the number of points received, depending on the statistics of the played matches and goals.
